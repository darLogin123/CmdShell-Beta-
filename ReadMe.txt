------------------------------
Description
------------------------------
CMD Shell v1.13 is a custom command shell
made with Windows batch script.

It supports built-in commands and
native Windows CMD commands.

------------------------------
Features
------------------------------
- Custom CMD shell interface
- Color command support
- Echo command support
- Start / Open App or Website
- Supports all CMD commands (dir, ipconfig, etc.)
- Lightweight and simple

------------------------------
Commands
------------------------------
help        - show help page
clear       - clear screen
ver         - show version
whoami     - show username
color XX    - change console color
echo text   - print text
open app    - Start App, website
start app   - native CMD start
exit        - exit shell

------------------------------
Examples
------------------------------
color 0a
echo Hello World
open notepad
open https://google.com
dir /s

------------------------------
How to Use
------------------------------
1. Double click cmdshell_v1.13.bat
2. Type commands in the shell
3. Use "help" to see command list
